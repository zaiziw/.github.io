unity学习笔记 roll-a-ball

移动玩家

移动

  1. 首先获得键盘输入

    float moveHorizontal = Input.GetAxis("Horizontal");
float moveVertival = Input.GetAxis("Vertical");

    通过GetAxis获得的Horizontal和Vertical都是来自unity的设置

  2. 绑定刚体

    rb = GetComponent<Rigidbody>();

  3. 让刚体移动

    Vector3 movement = new Vector3 (moveHorizontal, 0.0f, moveVertival);
rb.AddForce(movement);

    公共变量

script里面的公共变量会出现在Unity的Inspector里面

移动摄像机

  1. 获取物品的位置
    transform.posion
    类型为vector3

  2. Update

    1. Update

      跟平台帧数相关,用于非实体的更新

    2. FixedUpdate

      • 是真实时间
      • 实体的更新必须用FixexUpdate

    3. LateUpdate

      • 最后进行的update
      • 相机的update可以用在这里

        设置收集物

        deltaTime

        距上一次调用Update或FixedUpdate所用的时间
        使变化效果依赖于单位时间而不是帧间隔

        prefabs

        对预设资源进行的改变会运用到每一个实例上

        碰撞器

      void OnTriggerEnter(Collider other)
      检测到碰撞事件的时候会调用它

Colider other是被碰到的物体

other.gameObject.CompareTag()

用来区分物体

需要设置Is Trigger来保证可以穿过它

缓存

静态碰撞器在运动时都会重新缓存所有静态碰撞器

  • 带有碰撞器和刚体的组件是动态的
  • 带有碰撞器但是没有物理刚体的是静态的

    刚体

  • Use Gravity

    不受重力影响但是会受其他物理力影响

  • is Kinematic

    可以运动但是不会受物理力影响，所以可以作为事件的触发器
