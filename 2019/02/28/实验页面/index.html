Hexo使用记录

部署页面三步骤

清理缓存

hexo clean

进行渲染

hexo generate

可以简写成 hexo g

部署到本地

hexo server

(调试使用) 可以简写 hexo s。

然后浏览器输入 http://localhost:4000

注意调试完之后 control + C 关闭本地端口

部署到git服务器

hexo deploy

调试完毕后使用 hexo deploy 简写为 hexo d来部署到git服务器。

编辑文章

新建页面

hexo new "title"
